## 使い方

### 1. リポジトリのクローン
```
git clone https://github.com/kentakom1213/Animaker.git
```

### 2. モジュールのインストール
```
pip install -r requirements.txt
```

グローバル環境にインストールするのが良くない場合は、venvなどの仮想環境を使いましょう。

### 3. コードの作成

[example](example/)のコードを参考に、好きなアニメーションを作ってみましょう。
